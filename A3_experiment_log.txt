================================================================================
  MODELS
================================================================================

1. MF (Matrix Factorization)
   Loss     : BPR + L2 regularisation
   Params   : User/item embeddings only

2. LightGCN
   Loss     : BPR + L2 regularisation
   Encoder  : 3-layer graph propagation + mean pooling
   Ref      : He et al., SIGIR 2020

3. RecDCL (Assignment 2 baseline)
   Loss     : BPR + λ_BCL×BCL + λ_FCL×FCL + L2
   λ_BCL    : 0.05 (fixed)
   λ_FCL    : 0.05 (fixed)
   Ref      : Zhang et al., WWW 2024

4. AdaptRecDCL (Proposed Method)
   Loss     : BPR + λ_BCL(t)×BCL + λ_FCL(t)×FCL + L2
   λ values : Learnable via GradNorm (Chen et al., ICML 2018)
   Init     : λ_BCL = λ_FCL = 0.05 (same starting point as RecDCL)
   Adapt LR : 0.01 (separate optimiser for λ parameters)
   GradNorm : α=1.5, updated every 5 training steps

================================================================================
  ANALYSIS OF RESULTS
================================================================================

1. LightGCN performs best on Gowalla (0.0962 vs 0.0662 for MF, +45.3%)
   On Amazon Beauty, LightGCN and MF are essentially tied (0.2334 vs 0.2328).
   This confirms that graph structure helps more on sparser data.

2. RecDCL underperforms MF and LightGCN on both datasets.
   On Gowalla: RecDCL = 0.0537 vs MF = 0.0662 (-18.9%)
   On Beauty:  RecDCL = 0.1929 vs MF = 0.2328 (-17.1%)
   This is consistent with the sensitivity issue identified in Assignment 2 —
   the fixed λ=0.05 causes the contrastive losses to misalign with BPR on
   these specific dataset/subset configurations.

3. AdaptRecDCL performs comparably to RecDCL on both datasets.
   λ_BCL reduces slightly (~5-19% drop from 0.05) while λ_FCL remains stable.
   The GradNorm signal is too weak at these λ scales to drive significant
   rebalancing — the gradient norms of all three tasks are already close in
   magnitude when λ=0.05, so GradNorm makes only minor corrections.

4. The lambda evolution plots confirm the GradNorm mechanism is functioning
   correctly (λ_BCL decreases because BCL gradient grows relative to BPR as
   training progresses), but the magnitude of adaptation is small (0.04-0.05 range).
   A larger initial λ mismatch would produce more dramatic GradNorm behaviour.

5. Amazon Beauty shows MF still improving at epoch 300 — this dataset benefits
   more from extended training than graph/SSL methods, likely because the
   interaction graph is dense enough that BPR already captures the key signals.
